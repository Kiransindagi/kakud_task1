# NBRO Landing Page

This is the landing page for the National Bio Research Organisation (NBRO), a unified ecosystem where learning, research, industry, and innovation meet.

## Getting Started

### Prerequisites

- Node.js (v18 or later recommended)
- npm (comes with Node.js)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Kiransindagi/kakud_task1.git
    cd kakud_task1
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

To start the local development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

## Technologies Used

-   [Next.js](https://nextjs.org/) - React Framework
-   [React](https://reactjs.org/) - UI Library
-   [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
-   [Lucide React](https://lucide.dev/) - Icons

## Project Structure

-   `app/`: App router pages and global styles.
-   `components/`: Reusable UI components (Hero, Navbar, Stats, etc.).
-   `public/`: Static assets (images, icons).
